# AtomWorld-Mirror

长时程原子模拟的逐步更新会把大量预算耗在低影响中间态上，而 AtomWorld-Mirror 学习关键演化主干上的可达宏观状态-时间转移，在压缩微观事件回放的同时保持结构有效性、数量守恒和物理时间一致性。

## 项目概览

AtomWorld-Mirror 采用“教师模拟器 + 学生潜在世界模型”的结构。

- 教师模拟器提供原子构型、局部可达支持、路径摘要和物理时钟监督。
- 学生潜在世界模型学习受控步长集合上的多 K 半马尔可夫宏步转移，例如 `K = {2, 4, 8}`。
- 模型输出是候选位点上的稀疏编辑，而不是整张晶格的稠密重建。
- 训练和推理都经过同一条带投影的前向路径，使闭环展开看到的是一致的受约束状态。

## 核心能力

- **可达稀疏编辑**：宏步预测被限制在局部演化能够支持的候选集合内。
- **物理时间建模**：每个宏观转移预测累计期望物理时间，而不只是状态端点。
- **多 K 步长**：模型可以在受控宏步步长中选择，而不是固定单一步长。
- **投影闭环展开**：数量守恒和局部可达约束进入前向路径，而不是只作为后处理。
- **训练-推理一致的路径表示**：训练期使用路径摘要，推理期使用学到的先验，避免训练期偷看未来路径。

## 物理约束

方法必须同时满足三条物理承诺：

| 约束 | 含义 |
| --- | --- |
| 局部可达性 | 预测编辑必须落在可达候选支持内。 |
| 数量守恒 | 投影后原子和空位数量保持一致。 |
| 连续时间一致性 | 宏步时间对应教师路径上的累计物理时钟。 |

## 安装

公开复现建议使用 Python 3.10 或 3.11。

```bash
python -m pip install --upgrade pip
python -m pip install -r requirements.txt
```

如果需要特定平台的加速库，请先安装匹配的 PyTorch 版本，再执行同一个 `requirements.txt` 安装命令。更具体的依赖说明见 [`dependencies.md`](dependencies.md)。
